ROC Curve:¶

Receiver Operating Characteristics(ORC) curve displays the classification capabilities of a classifier system as threshold is varied. We Plot True Positive Rate vs False positive rate where:
True positive rate = TP / TP+FN = Recall
False Positive rate = FP / FP+TN

Since we are just comparing within a single positive and negative label, if the classes are imbalanced, there will not be any issue.
The ROC curve provides us with an opportunity to compare the performance of the model and also compare between various models.s

Lift Curve¶

It is the measure of a targeting model at predicting cases as having an enhanced response measured against a random choice model. It is a measure of how much does a model perform betternthan a random targetting model.
in the above case, the Lift curve shows how much better our model performs at predicting both class1 and class 0 compared to a random scenario.

ROC Curve:¶

Receiver Operating Characteristics(ORC) curve displays the classification capabilities of a classifier system as threshold is varied. We Plot True Positive Rate vs False positive rate where:
True positive rate = TP / TP+FN = Recall
False Positive rate = FP / FP+TN

Since we are just comparing within a single positive and negative label, if the classes are imbalanced, there will not be any issue.
The ROC curve provides us with an opportunity to compare the performance of the model and also compare between various models.s

Lift Curve¶

It is the measure of a targeting model at predicting cases as having an enhanced response measured against a random choice model. It is a measure of how much does a model perform betternthan a random targetting model.
in the above case, the Lift curve shows how much better our model performs at predicting both class1 and class 0 compared to a random scenario.
